*A 2D arcade-style game built in Processing (Java Mode)*

**AquaLava Escape** is a lightweight, scene-based 2D game developed entirely in [Processing 4](https://processing.org/).  
You control a character across multiple environments, facing enemies, unlocking levels, and exploring a pixel-art world inspired by classic arcade design.
